# Flowers — интернет-магазин букетов

Flowers — это учебный командный pet-проект веб-приложения для продажи букетов и управления заказами.  
Проект имитирует работу интернет-магазина: пользователь может просматривать каталог букетов, выбирать товары, добавлять их в корзину и оформлять заказ. Также в системе предусмотрена административная часть для управления товарами и заказами.

## Моя роль в проекте

В этом проекте я занимался frontend-разработкой.

С моей стороны было реализовано:

- верстка пользовательского интерфейса;
- создание страниц и компонентов приложения;
- отображение каталога букетов;
- работа с корзиной и заказами;
- подключение frontend к backend через HTTP-запросы;
- обработка данных, полученных с сервера;
- участие в командной разработке и работе с Git.

## Основной функционал

- просмотр каталога букетов;
- фильтрация/разделение товаров по категориям;
- добавление товаров в корзину;
- оформление заказа;
- отображение информации о товарах;
- административная панель для управления данными;
- взаимодействие frontend и backend через API.

## Технологии

## Цель проекта

Цель проекта — получить практический опыт командной разработки полноценного веб-приложения, научиться связывать frontend с backend, работать с данными и создавать удобный интерфейс для пользователя.

## Статус проекта

Проект выполнен в учебных целях и может использоваться как pet-проект в портфолио frontend-разработчика.
